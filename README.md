# Python CAD Devcontainer

Devcontainer setup for CAD modeling with [build123d](https://github.com/gumyr/build123d) and [ocp-vscode](https://github.com/bernhard-42/vscode-ocp-cad-viewer).

The [intro_examples](./intro_examples/) folder contains the builder mode code from the
[Introductory Examples](https://build123d.readthedocs.io/en/latest/introductory_examples.html) section of the official build123d documentation.

The [tutorials](./tutorials/) folder contains the tutorials from the
[Tutorials](https://build123d.readthedocs.io/en/latest/tutorials.html) section
of the official build123d documentation.
